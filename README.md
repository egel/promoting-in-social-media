## Convert to .doc or .odt

Install for example `pandoc` program

```
$ sudo apt-get install pandoc
```

Then convert your file by:

```
$ pandoc -s document.tex -o document.odt
```

or
```
$ pandoc -s document.tex -o document.docx
```
